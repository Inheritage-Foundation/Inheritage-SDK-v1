# Example Gallery

Four end-to-end samples showcasing the Inheritage SDK in different environments.

| Example | Description |
| ------- | ----------- |
| [`nextjs-map`](./nextjs-map/) | Interactive map plotting GeoJSON features for rapid mapping apps. |
| [`react-gallery`](./react-gallery/) | Media gallery powered by React Query and the `/media` endpoint. |
| [`ai-similarity-demo`](./ai-similarity-demo/) | Command-line recommendation helper using deterministic embeddings. |
| [`api-fetch-script`](./api-fetch-script/) | Node.js export script for researchers and data journalists. |

Each folder includes a self-contained README and starter code. Copy, tweak, and share what you build in [Discussions](https://github.com/0xMilord/Inheritage-SDK-v1/discussions).
